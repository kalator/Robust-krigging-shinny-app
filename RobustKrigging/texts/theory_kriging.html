Kriging

Kriging is a geostatistical algorithm of interpolation that is based on weighted mean with optimized weights to minimize the error of estimate. Variogram is used for retrieving such weights. Method of kriging was developed in geology and it is suitable to use in cases, where a spatial variable is modified within space with certain continuity (e.g. observed values of maximum temperature are more similar at points that are closer to each other) but it is not possible to describe such variable with simple function.

There are several types of kriging. A basic method of kriging is called Ordinary kriging (OK). This method is considering unknown expected value that is constant in whole area. The value of predicted variable z* at point x0 in space is computed using values from surrounding points z(xi) and their weights λi: $$z^*(x_0) = \sum \lambda_i z(x_i)$$
Let's compare this method with e.g. the approach of interpolation method IDW (inverse distance weighting), where weights are calculated as inverted value of distances (or powered distances). This method doesn't consider mutual configuration of data points. The aim of kriging is to eliminate this drawback.
There are two conditions for weights in kriging:

  • The expected value must be zero, therefore sum of weights must equal 1: $$\sum \lambda_i = 1$$

  • The variance of estimate of the correct value is minimal: $$\sigma^2 = var[z(x_0)-z^*(x_0)] = min$$

With further adjustments we will get Kriging equation system:

$$\sum_j \lambda_j \gamma(x_i-x_j) + \mu_L = \gamma(x_i-x_0),i=1,2,...,n$$ $$\sum_j \lambda_j = 1$$

where μL is Lagrange multiplier. The equation system has n+1 equations for n+1 unknown variables (weights and Lagrange multiplier).
