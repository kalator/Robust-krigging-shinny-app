Purpose of this web application is to comprehensibly present the basics of kriging, a method of interpolation that uses geostationary methods of estimate. The application is written in R using Shiny package. Shiny is an open source package that provides web framework for building web applications using R. There are few interactive widgets alowing user to make some computation himself. Computations are processed with R packages geoR and georob with data that are provided natively by these packages.
